Создание сайта


Перед установкой фреймворка foundation 6 необходимо установить (если они у вас не установлены) следующие программы:
1. Система контроля версий GIT
2. Програмная платформа Nodejs
 Далее переходим на сайт http://foundation.zurb.com/ в раздел загрузки или документации(Docs) и устанавливаем фреймворк foundation 6. Для этого копируем строку npm install --global foundation-cli, переходим в GIT BASH и запускаем установку. После завершения установки foundation cli открываем GIT BASH и переходим к созданию нового проекта. Cоздаём папку на любом диске для проектов и возращаемся в GIT BASH. Нужно зайти в папку проекта. Для этого прописываем cd d:/(или c:/ в зависимости от диска), нажимаем Enter, потом cd название папки. Потом вставляем командную строку: foundation new --framework sites --template zurb. Далее следуем указаниям установщика, а именно задаем имя своего проекта (обязательно без пробелов). У вас на диске будет создана папка с названием проекта и в эту папку будут скопированы все необходимые файлы. После завершения установки в терминале нужно перейти в папку вашего проекта и выполнить команду npm start. Начальная страница будет запущена в вашем браузере
